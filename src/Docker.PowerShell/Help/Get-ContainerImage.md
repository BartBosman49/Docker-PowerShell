# Get-ContainerImage
## SYNOPSIS
Get-ContainerImage \[-All\] \[-HostAddress \<string\>\] \[-CertificateLocation \<string\>\] \[\<CommonParameters\>\]
## SYNTAX

### Default (Default)

Get-ContainerImage [[-Id] <String[]>] [-HostAddress <String>] [-CertificateLocation <String>]
 [<CommonParameters>]

## DESCRIPTION
Returns container images.
